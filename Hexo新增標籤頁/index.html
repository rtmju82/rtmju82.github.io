Hexo新增標籤頁

發表於 2023-11-27

Hexo使用Next主題後預設就會將文章和標籤顯示在頁面上，但是實際要使用的時候會發現…

怎麼不能點？？(滑鼠移到標籤的時候並不會出現手指)

原來是還要經過一點小小的設定才可以順利使用標籤，接著就讓我們來設定吧~

建立文章標籤頁面

這邊我目前只有下hexo new page tags的指令

執行完後就會看到source/底下多了一個tags的資料夾

我們要到裡面去編輯index.md這個檔案，加上

type: tags

這邊做完之後就可以再重新產生靜態網頁並且Run服務，就會發現標籤頁可以點選囉~

利用主選單新增標籤頁面

原本的標籤連結已經可以點選了，但是如果想要將標籤這個項目獨立出來的話，可以按照下面的方式修改

這邊以Next主題為例，所以我們要到 ./themes/next/_config.yml去更改設定，可以看到原本Tags是被註解起來的

我們將#移除，接著重新部署並Run服務，就完成囉~
